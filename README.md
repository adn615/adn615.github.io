## ğŸ”§ GitHub Pages Deployment

1. **Fork/Clone this repository** to your GitHub account
2. **Rename the repository** to `yourusername.github.io`
   - Replace `yourusername` with your actual GitHub username
3. **Customize the content**:
   - Edit `index.html` with your personal information
   - Update contact details, project descriptions, and links
   - Replace placeholder images with your actual portfolio work
4. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click "Save"
5. **Access your site** at `https://yourusername.github.io`

## ğŸ¨ Customization Instructions

### Personal Information
Update the following in `index.html`:
- Name, title, and description in the hero section
- About section content and statistics
- Skills and education details
- Experience section with your internships and jobs
- Contact information (email, phone, location)
- Social media links

### Resume Download
Add your resume PDF to the root directory as `resume.pdf`:
- File should be named `resume.pdf` exactly
- Keep the file size reasonable (under 5MB) for web loading
- The download button in the hero section will work once you add this file

### Styling
- Colors and fonts can be modified in `styles/styles.css`
- Layout adjustments in `styles/main.css`

### Portfolio Images
Replace the Unsplash URLs in `index.html` with your own image URLs. Recommended image size:
- Hero profile image: 400x400px
- Portfolio images: 600x400px

### Adding More Projects
Duplicate a `<div class="portfolio-item">` in the portfolio grid and update:
- Image source
- Title and description
- Category tag
